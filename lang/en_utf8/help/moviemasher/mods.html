Moviemasher

 

Deployment instructions:

extract moviemasher mod into moodle mods dir

goto moodle/admin to setup tables

install red5 and ffmepg, if not previosly installed

create a link inside red5/webapps to your <moodledir>/mod/moviemasher/Recorder/web and name it "recorder"

give red5 write permission over <moodledir>/mod/moviemasher/Recorder/web/streams directory

have fun :)
